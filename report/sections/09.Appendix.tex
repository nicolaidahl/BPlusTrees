\section{Running the code}
The code for this project is hosted on Github at \url{https://github.com/nicolaidahl/BPlusTrees}. The Coq implementation resides in the ./code/ subdirectory. A Makefile is provided that will build the project, assuming Coq 8.3 or later is installed.
